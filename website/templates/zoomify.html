How to use StoryMap with images

In addition to using StoryMap to tell a story about places in the world, our tool can be used to guide people through a closer look at images: works of art, historical maps, photography, or anything else you can think of.

A Simple Tutorial

This tutorial assumes the following:

  • You have Adobe Photoshop with the Zoomify plugin installed. The plugin is installed by default in Photoshop CC.
  • You have a very large imageâ€”this method is only interesting if your image is at least 8-10,000 pixels on at least one dimension. (Below we'll use this image from Wikimedia commons.)
  • Access to a web server where you can share an entire folder of images. (The instructions below will explain how to do this with Dropbox.)

Getting started
  1. Open your image in Adobe Photoshop
  2. Note the pixel dimensions of your image:
    1. From the Imagemenu, choose Image Size
    2. If the dimensions aren't in pixels, use the menu next to "Dimensions:" to set it to pixels.
    3. Make a note of the dimensions: the first number is width, the second is height.
  3. From the File menu choose Export > Zoomify.
  4. Set the output folder and click OK. You can ignore the other settings.
  5. After you click OK, Photoshop will open a page in your browser showing your Zoomify export. You can ignore this.
  6. Put the folder of images which you exported on a web server and make note of the root URL. Here's how to do that using DropBox:
    1. Upload the entire folder (from step 4 above).
    2. In the folder you uploaded is another folder, which should have a name like the original file name, except without an extension. Open that folder.
    3. Locate a file named ImageProperties.xml and copy its public link.
    4. Paste it somewhere for a minute, and delete the "ImageProperties.xml" part of the URL. For example: copied: https://dl.dropboxusercontent.com/u/2653407/adams_panorama_tiles/1878_Adams_Monumental_Illustrated_Panorama_of_History_-_Geographicus_-_WorldHistory-adams-1871/ImageProperties.xml edited: https://dl.dropboxusercontent.com/u/2653407/adams_panorama_tiles/1878_Adams_Monumental_Illustrated_Panorama_of_History_-_Geographicus_-_WorldHistory-adams-1871/
  7. Go to http://storymap.knightlab.com/select.html and click on "New Zoomify StoryMap".
  8. Enter a name for your StoryMap, the URL (from step 6.4) and the image dimensions (from step 2.3), and click "Create".

From here on out, there's not much different than making a normal StoryMap. Of course, you can't search for places to put your pins on the map. Instead, navigate to the map and double-click on the location where you want your pin. You can also drag an existing pin to a new location.

We're still refining this feature, so let us know what you think and how we could improve it.
